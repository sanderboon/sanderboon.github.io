Sander Boon

My name is Sander Boon, I'm an experienced front-end engineer, that loves to play with HTML.

About

I'm a full-time front-end engineer, currently employed at Be Informed. I have over 15 years of experience working on all kind of IT related projects. I started working at DataSmit BV, creating their e-commerce and auction products. Then I spent a couple of months working at Twice (now Compu'Train), maintaining their software and creating their website, and as said, currently working at Be Informed, doing all kind of things from supporting projects by building a customized front-end experience to creating the default web interface of the bpm suite.

Working on large innovative projects is what I like to do and that's what I'm doing best. Trying to keep up with the latest and greatest web technologies, implementing what the client wants, making it responsive, accessible and deliver it testable and performant is what I want to do.

Using html5 when applicable, using xhtml 1.0 when required. I can write plain JavaScript, but have no problem using libraries and frameworks like jQuery, prototype, ReactJS, et cetera. I'm used to write my css without pre processors, but have no problem creating it with LESS or SASS. I work in an environment where we use Scrum as our agile methodology, we set up our unit testing framework using Jasmine.

Xslt is what we used to work with at the companies I've worked for, so I'm certainly a master in that (xslt 1.0 ans 2.0 to be precise, yes I know how to use the Muenchian Method). At DataSmit I wrote code in classic ASP 3.0 and later on with asp.net, which I continued doing at Twice, but this has been some time ago. SQL server, mySQL, H2 and oracle are databases I've worked with. As a side project I used to do some PHP (old 3 and 4). I have worked with systems as Wordpress, Hippo and other open source projects.
